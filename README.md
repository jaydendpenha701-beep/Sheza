# Shadows of the Abandoned - 3D Horror Game

A complete 3D first-person horror game built with Three.js.

## Features
- First-person perspective with FPS controls (WASD + Mouse)
- Modular environment with custom "dirty" textures
- AI Enemy with Patrol and Chase states
- Interaction system for collecting items
- Dynamic flashlight with flickering effect
- Spatial audio (synthesized)
- HUD and Menu systems (Main, Pause, Game Over, Win)

## How to Run
1. Ensure you have a local web server (e.g., `python3 -m http.server` or `live-server`).
2. Open `index.html` in a modern web browser.
3. Click "START GAME" and enjoy!

## Controls
- **W, A, S, D**: Move
- **Shift**: Run
- **Ctrl**: Crouch
- **Mouse**: Look around
- **Click or E**: Interact / Collect items
- **Esc**: Pause

## Objective
Collect 5 keys hidden throughout the map and find the exit (starting position) to escape. Avoid the monster!
